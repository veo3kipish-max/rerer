# ⚠️ ВАЖНО: Переназначение домена kipish.fun

## Проблема
Домен `kipish.fun` уже привязан к другому проекту в Vercel. Чтобы использовать его для этого приложения, нужно сначала отвязать его от старого проекта.

## Шаги для переназначения домена

### 1. Откройте страницу Domains
Я уже открыл страницу в браузере:
https://vercel.com/nicks-projects-00786ef7/1123/settings/domains

### 2. Найдите старый проект с kipish.fun
1. Откройте новую вкладку: https://vercel.com/dashboard
2. Найдите проект, к которому привязан `kipish.fun`
3. Перейдите в Settings → Domains этого старого проекта
4. Найдите `kipish.fun` в списке доменов
5. Нажмите на кнопку удаления (три точки → Remove)

### 3. Добавьте домен в текущий проект
После удаления из старого проекта:
1. Вернитесь на страницу: https://vercel.com/nicks-projects-00786ef7/1123/settings/domains
2. В поле "Add domain" введите: `kipish.fun`
3. Нажмите кнопку "Add"
4. Подождите несколько минут для активации SSL

### 4. Настройте DNS (если еще не настроено)
Добавьте в настройках вашего регистратора доменов:

**Рекомендуемый способ (CNAME):**
```
Type: CNAME
Name: @
Value: cname.vercel-dns.com
```

**Альтернатива (A-запись):**
```
Type: A
Name: @
Value: 76.76.21.21
```

### 5. Обновите ссылки
После успешного добавления домена:

**В Telegram боте (@BotFather):**
```
/mybots → выберите бота → Bot Settings → Menu Button
URL: https://kipish.fun
```

**В Supabase Secrets:**
```
WAYFORPAY_DOMAIN=kipish.fun
```
