# Быстрая настройка Supabase

## 1. Выполнить SQL схему

Откройте: https://supabase.com/dashboard/project/ndrdksmdkhljymuvxjly/sql/new

Скопируйте весь файл `supabase/schema.sql` и выполните в SQL Editor.

## 2. Проверка

После выполнения SQL у вас будут созданы таблицы:
- ✅ users
- ✅ payments  
- ✅ generations
- ✅ subscription_plans (с данными)
- ✅ credit_packages (с данными)
- ✅ admin_logs

## 3. Готово!

База данных настроена и готова к использованию.
